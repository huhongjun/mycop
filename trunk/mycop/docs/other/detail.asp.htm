电子政务协同工作公共平台集成件解决方案

    电子政务协同工作公共平台集成件解决方案

　 1、集成件
    基于IBM、GDCA、Gever(金宇恒科技)软、硬件的最优整合,实现"一体化政务"的运行模式，为基层政府提供先进、高效、安全的解决方案。
2、集成件特点
    一站式政务协同平台
    安全可靠
    技术先进
    规划全面、功能强大
    安装部署简单，用户维护量比较少
3、 产品介绍                                                                 
    专注于基层电子政务，为基层政府提供全面、及时的解决方案。
    平台以工作流为核心，以人为本，以日常办公、协同工作、事务管理、组织管理、信息中心等业务为基础，以知识为源泉，真正实现全员协同办公。
    平台功能强大、技术先进、安全可靠。产品以服务为导向、以企业和市民为中心、通过多种个性化人性化设置，实现一站式行政审批及协同办公。同时平台具有跨操作系统、跨数据库、跨应用服务器等特点。

4、 技术特点                                                                         
    Ø    基于J2EE的软件架构
    平台采用J2EE的软件体系框架，构建出一个多层的、具有良好的扩展性、移植性、可伸缩性。
通过采用先进的MVC软件开发模式，复用大量我公司自主开发并经过严格测试的电子政务类库，保证了平台的稳定性与健壮性。
    Ø    平台独立无关性
    本平台充分考虑了各种操作系统，应用服务器和数据库服务器，程序设计中采用标准语法编写。实现了平台独立无关性。支持跨平台、跨数据库、跨应用服务器。
    Ø    支持CA身份认证、电子签名技术、电子印章
    平台全面支持电子政务统一安全平台，实现了与SWP03公钥基础设施安全平台的CA安全认证的接口。通过CA安全机制来保证系统的安全可靠性，通过一人一卡来保证安全登录和用户操作的不可撤销、不可抵赖性。
    Ø    扩展的Struts架构
    通过对开放源码Struts进行的扩展，可以灵活的实现系统的翻页、权限、框架等功能，具有先进性及高维护性的特点。
    Ø    基于Web Service信息交换技术
    通过采用Web Service实现了各个子系统之间的数据传输、数据共享，避免了信息孤岛的产生。
    Ø    数据库开发技术
     采用IBM DB2 UDB Enterprise V8.1数据库设计方案，系统既具有高性能的适应联机事务处理应用、良好的可扩展性及海量存储功能，又实现各类文档高效、可靠的流转，并通过异构型数据库 的有机结合，实现了办公与业务数据的融合与共享。
5、功能简介                                                           

6、 功能模块                                                   
    电子政务综合平台按功能可分为：办公管理、业务管理、公众服务、系统管理四个子系统。
    Ø    办公管理
     办公管理子系统主要对基层政府日常办公和内部信息流转进行管理，提供了包括日常办公、公文处理、事务管理、知识管 理、组织管理、档案管理、信息中心等协同办公，同时包括邮件管理、手机短信、会议管理、督办管理、信访管理等功能。
    Ø    业务管理
    业务管理子系统涵盖了基层政府所需处理的大部分业务信息，主要包括人口、计生、组织、社务、共青团、妇联、信访、财政、综治、教育、旅*、工业、农林水、人大、宣传、纪监、司法、武装等子模块。
    Ø    公众服务
     政府一站式公众服务是为本地区、本部门的政府人员、服务人员、媒体发布者、企事业、公民提供的一个统一互联网访问入 口和集中管理控制平台。对内可以实现信息沟通，系统工作和资源共享，解决由于多种资源并存、多种体系独立而造成的信息沟通不畅的问题；对外实现行政审批中 心，政务公开、政策宣传、招商引资等业务。其主要包括政务中心、办证中心、呼叫中心、政府商务、政务公开、政府公告、政策法规等业务。
    Ø    系统管理
    对平台的整个组织架构、岗位、人员、权限以及相关的基础数据、业务流程进行维护管理。功能主要包括组织管理、用户管理、角色管理、流程管理、数据字典、日志管理、数据备份与恢复等。
7、 产品特点                                                   
    Ø    公文流转自动化
    采用自主知识产权的可视化自定义工作流引擎，用户通过流程设置对各流转环节进行完全的自定义，实现了各类公文流转和内部审批。
    Ø    一站式政务平台
     系统通过互联网实现审批的全过程，整合政府职能、优化业务流程，构建标准统一、安全可靠的政务应用服务平台，将各类 办公申请交换到各部门内部办公平台进行业务处理，实现跨部门的协同办公、信息共享及公共服务一体化。协助政府提高办事效率和透明度，创造良好的投资环境， 使政府从“管理型政府”向“服务型政府”转变，更好的为人民服务。
    Ø    以服务为导向
    通过建立公众服务窗口，使政府面向公众、面向企业。加强政府与公众的信息交流，实现以人为本，增强为公众“服务”的意识。体现务实政府、效率政府、施政为民的理念。
    Ø    以企业和市民为中心
     构建以企业和市民服务为中心、充分便民利民、平台不仅有效保证了电子政务网上服务信息流的有效运转，同时为其它业务 的运转提供了良好的窗口环境和通道，使不同层次的社会公众都可以通过适合自己的方式获取有效的电子政务服务和反馈。
    Ø    个性化、人性化
    平台建设充分考虑系统的个性化、人性化。通过一站式的协同政务,实现部门与部门、部门与个人、公务员与公务员的协同工作，减少了重复劳动。通过资源的有效整合,加快了对信息的处理，提高了工作效率和工作质量。
    办公、协同工作、事务管理、组织管理、信息中心等业务为基础；以知识为源泉，真正实现全员协同办公。
    平台功能强大、技术先进、安全可靠。产品以服务为导向、以企业和社民为中心、通过多种个性化人性化设置，实现一站式行政审批及协同政务。
8． 集成件软硬件配置                                           
    Ø    软件配置
    集成件应用平台：GEVER电子政务协同工作公共平台
立 足于基层电子政务，为镇区电子政务的建设提供先进、可靠、高效的服务。通过政务平台使政府部门能够及时处理公文信息；加强对突发事件的反应能力，加快政府 内部信息的流转、业务的处理、部门的协调；全面提升政府部门的工作效率及服务水平，以便更好地为企业，为社民服务，使其成为政府的执政工具。
    集成件操作系统：Red hat AS/红旗Linux
专为企业关键业务提供服务的Linux 解决方案。适合运行数据库，中间件和集群/负载均衡系统等应用，支持各种平台的服务器。
    集成件数据库：IBM DB2 Universal Database™
提供随着业务需要（既包括成本方面，也包括功能方面）而向上或向下扩展的能力，还可以减少和简化IT资源上的负担，从而可以利用令人激动的新技术来帮助企业无需关注数据管理。
    集成件中间件：IBM WebSphere
提供了一个具有高级性能和可用性的操作环境，用来支持动态应用程序环境。除了基本 WebSphere Application Server 内的所有特性和功能外，此配置还提供高级部署服务，包括群集、网络边界服务、Web 服务增强功能和分布式配置的高可用性。
    Ø    硬件配置
    服务器：IBM OpenPower
     适用于 Linux® 操作系统，并构建于开放式 Linux 操作系统的标准，提供卓越的性能和工作效率，为入门级的服务器带来卓越的功能，诸如 IBM Virtualization Engine™ 技术以及“可靠性、可用性和可维护性（RAS）”能力，从而帮助提高系统利用率、减少宕机时间并降低系统管理成本
    Ø    GDCA认证
      数字证书
      安全服务中间件
      可信证据服务器
      各种业务扩展型证书
      安全应用支撑服务器
      资源访问认证服务器
      电子印章
